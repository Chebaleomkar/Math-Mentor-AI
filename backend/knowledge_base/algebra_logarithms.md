# Logarithms and Exponential Equations

## Key Properties of Logarithms
Let $b > 0, b \neq 1$, and $x, y > 0$:
1. **Product Rule**: $\log_b(xy) = \log_b(x) + \log_b(y)$
2. **Quotient Rule**: $\log_b(\frac{x}{y}) = \log_b(x) - \log_b(y)$
3. **Power Rule**: $\log_b(x^k) = k \cdot \log_b(x)$
4. **Change of Base**: $\log_b(x) = \frac{\log_c(x)}{\log_c(b)}$ (usually $c=10$ or $c=e$)
5. **Inverse Properties**: $b^{\log_b(x)} = x$ and $\log_b(b^x) = x$

## Solution Strategy for Exponential Equations
1. **Isolate the Exponential Term**: Get the term with the variable exponent by itself.
2. **Take the Logarithm**: Apply $\ln$ (or $\log$) to both sides of the equation.
3. **Apply Power Rule**: Bring the exponent down.
4. **Solve for Variable**: Isolate the target variable.
5. **Check Domain**: Logarithms are only defined for positive arguments. Always check if the solution works in the original equation!

## Common Mistakes
- **Domain Errors**: Forgetting that $\log_b(x)$ requires $x > 0$. **Always verify solutions.**
- **False Rules**: Assuming $\log(x+y) = \log(x) + \log(y)$ (This is FALSE!).
- **Base Confusion**: Mixing up natural log ($\ln$, base $e$) and common log ($\log$, base 10) without explicitly converting.
